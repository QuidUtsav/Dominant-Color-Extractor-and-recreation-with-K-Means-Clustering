# Dominant-Color-Extractor-and-recreation-with-K-Means-Clustering
Extracts dominant colors from any image using K-Means clustering (SciPy) and visualizes results with color quantization.

Features:
- Loads and processes RGB pixels from JPEG/PNG images
- Applies K-Means to find dominant colors
- Displays original vs quantized (posterized) image side-by-side
- Shows top dominant color swatches
